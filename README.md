## How to run

- clone this repo
  > `cd karem-omary-09-01-2020` 
  > `npm install` 
  > `cd fake-server` 
  > `npm install` 
  > `npm start`
- in another terminal
  > `cd karem-omary-09-01-2020` 
  > `npm start`
## for deployment:
- get an API key from accuweather
- add it as an enviroment variable "REACT_APP_APIKEY"
- emample: "REACT_APP_APIKEY = qH9VxSiDgm-Not_My_API_Key-ZLiAj2b7Fo7G"
